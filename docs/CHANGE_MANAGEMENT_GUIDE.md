### Common Resistance Patterns & Solutions

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        RESISTANCE PATTERNS & RESPONSES                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  PATTERN 1: "We don't have time for this"                                   │
│  ├─ Root Cause: Competing priorities, pressure to deliver                   │
│  ├─ Response Strategy:                                                       │
│  │  • Demonstrate quick ROI with time savings                               │
│  │  • Start with minimal time investment                                    │
│  │  • Show how it reduces current pain points                               │
│  │  • Executive messaging on priority                                       │
│  └─ Success Metrics: Participation rate, time-to-first-value              │
│                                                                               │
│  PATTERN 2: "The old system works fine"                                     │
│  ├─ Root Cause: Comfort with status quo, fear of change                     │
│  ├─ Response Strategy:                                                       │
│  │  • Quantify pain points of current system                               │
│  │  • Side-by-side comparison demonstrations                                │
│  │  • Allow gradual transition, not forced switch                          │
│  │  • Highlight early adopter success stories                              │
│  └─ Success Metrics: Voluntary adoption rate, satisfaction improvement     │
│                                                                               │
│  PATTERN 3: "This is just another tool that will be replaced"               │
│  ├─ Root Cause: Tool fatigue, past failed initiatives                       │
│  ├─ Response Strategy:                                                       │
│  │  • Executive commitment messaging                                        │

│  └─ Success Metrics: Sustained usage, community contributions              │
│                                                                               │
│  PATTERN 4: "This doesn't fit our specific needs"                           │
│  ├─ Root Cause: Unique requirements, customization needs                    │
│  ├─ Response Strategy:                                                       │
│  │  • Co-creation approach with user input                                  │
```

### Resistance Intervention Framework

```typescript
class ResistanceManager {
  async assessResistance(individual: TeamMember): Promise<ResistanceProfile> {
    return {
      level: this.calculateResistanceLevel(individual),
      type: this.identifyResistanceType(individual),
      rootCauses: this.analyzeRootCauses(individual),
      interventions: this.recommendInterventions(individual)
    };
  }
  
  private recommendInterventions(individual: TeamMember): Intervention[] {
    const interventions = [];
    
    if (individual.resistanceType === 'skeptical') {
      interventions.push(
        { type: 'evidence_based', action: 'Show concrete ROI data' },
        { type: 'peer_influence', action: 'Connect with early adopters' },
        { type: 'trial_period', action: 'Low-risk pilot participation' }
      );
    }
    
    if (individual.resistanceType === 'overwhelmed') {
      interventions.push(
        { type: 'support_intensive', action: '1:1 coaching sessions' },
        { type: 'gradual_introduction', action: 'Phased learning approach' },
        { type: 'time_management', action: 'Protected learning time' }
      );
    }
    
    if (individual.resistanceType === 'philosophical') {
      interventions.push(
        { type: 'vision_alignment', action: 'Connect to bigger purpose' },
        { type: 'choice_architecture', action: 'Emphasize voluntary adoption' },
        { type: 'co_creation', action: 'Involve in solution design' }
      );
    }
    
    return interventions;
  }
}
```
